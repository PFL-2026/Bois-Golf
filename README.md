# Bois Hit Dunas ⛳

An interactive golf scoring app for the **Dunas Course at Terras da Comporta** (par 71).
Two players, handicap-based shots given, metres→yards, stroke play **and** match play,
and a hole-by-hole photo gallery. Fully self-contained in a single `index.html`
(player photos are embedded) — works offline once loaded.

## Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g. `bois-hit-dunas`).
2. Upload the contents of this folder — `index.html` and `.nojekyll` — to the repo
   (drag-and-drop into the GitHub web uploader is fine).
3. In the repo: **Settings → Pages**.
4. Under **Build and deployment → Source**, choose **Deploy from a branch**.
5. Set the branch to **main** and folder to **/ (root)**, then **Save**.
6. Wait ~1 minute. Your app will be live at:
   `https://<your-username>.github.io/<repo-name>/`

## Use it like an app

Open the live URL on your phone, then **Share → Add to Home Screen**.
It launches full-screen and runs offline.

## Notes

- Scores, photos and notes are saved **locally on each device** (no cloud sync),
  so have **one phone be the official scorer** for the round.
- Tap any avatar to change a player photo; tap a frame in the **Holes** tab to add
  your own shot of each hole.
- All hole data is from the Dunas scorecard. Edit anything in the **Setup** tab.

Tee it high, let it fly.
